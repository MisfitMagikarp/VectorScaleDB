POST /v1/ingest
Batch ingest temporal vectors. Supports up to 10,000 vectors per request with automatic compression.
Request Body
{
  "vectors": [
    {
      "entity_id": "sensor-42",
      "entity_type": "LINK_METRIC",
      "vector": [0.1, 0.8, 0.3, 0.9],
      "timestamp": "2025-06-01T12:00:00Z"
    }
  ]
}
Response
{
  "ingested": 1,
  "compressed": 0,
  "elapsed_ms": 2.3
}
curl

  -d '{"vectors":[{"entity_id":"sensor-42","entity_type":"LINK_METRIC","vector":[0.1,0.8,0.3,0.9],"timestamp":"2025-06-01T12:00:00Z"}]}'
