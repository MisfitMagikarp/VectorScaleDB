Life Sciences

Bio-Computing & Neural Interfaces

Detect learning onset in neural organoids, correlate stimulus-response at millisecond precision, compare organoid behaviors via DTW, and orchestrate closed-loop control pipelines. From multi-electrode arrays to fleet-scale organoid ranking.

Multi-Electrode Array Recording Analysis

Ingest high-density MEA recordings at thousands of channels. Every electrode becomes a temporal vector stream with automatic spike detection and population state tracking.

Ingestion
Channel-Level Decomposition
MEA recordings from 64 to 4,096+ channels are decomposed per electrode. Each channel becomes a NeuralUnit entity with temporal vectors capturing firing rate, burst frequency, inter-spike interval distribution, and waveform shape. NWB and HDF5 formats ingested natively.
Population
Population State Vectors
Beyond individual channels, VectorScaleDB constructs PopulationState entities that capture the collective behavior of neural ensembles: synchrony index, spatial firing patterns, network burst rates, and functional connectivity matrices. Population states compress dramatically during stable activity periods.
Compression
Neural Regime Compression
Neural recordings exhibit long periods of baseline activity punctuated by bursts and state transitions. VectorScaleDB's drift-based compression captures this naturally: hours of baseline compress to single regime segments, while burst events and transitions retain full temporal resolution. 500x+ compression on continuous recordings.

Learning Onset Detection

Detect when an organoid begins learning. Change-point detection on compressed population state segments surfaces the exact moment behavior shifts from random to structured.

Detection
Change-Point Analysis on Neural Regimes
Learning onset manifests as a regime transition in the population state trajectory: firing patterns shift from stochastic to structured, synchrony increases, and stimulus-response latencies decrease. VectorScaleDB's change-point detection identifies these transitions automatically, timestamping the onset of learning to millisecond precision across all recorded channels simultaneously.
Quantification
Learning Rate Metrics
Once onset is detected, VectorScaleDB tracks the learning trajectory: how quickly does the population state converge to structured behavior? How stable is the learned regime? Does learning generalize across stimulation protocols? These metrics are derived directly from the compressed segment sequence, without re-processing raw recordings.

Stimulus-Response Correlation

Correlate stimulation events with neural responses at millisecond precision. Every stimulation becomes a temporal anchor for response analysis.

Precision
Millisecond-Resolution Alignment
StimulationEvent entities are ingested with precise timestamps. VectorScaleDB's temporal index allows querying "what was the population state in the 50ms window after each stimulation?" across millions of stimulation events. Response latency distributions, amplitude profiles, and spatial activation patterns are computed directly from the index.
Analysis
Response Trajectory Similarity
Compare neural responses across stimulation protocols using FastDTW. "Did the organoid respond to electrical stimulation at 10Hz the same way it responded to optogenetic stimulation at 10Hz?" Trajectory similarity scores quantify response equivalence across modalities, frequencies, and amplitudes.
Adaptation
Habituation & Sensitization Tracking
Track how responses evolve over repeated stimulation. VectorScaleDB's regime detection identifies habituation (decreasing response magnitude) and sensitization (increasing response magnitude) as behavioral regime transitions. Alert when an organoid's response profile shifts, indicating adaptation or fatigue.

Cross-Organoid Comparison & Fleet Ranking

Compare behavioral trajectories across organoids via DTW. Rank your entire fleet by learning speed, response consistency, and behavioral complexity.

Comparison
DTW Behavioral Comparison
Compare any two organoids' behavioral trajectories using FastDTW, accounting for temporal offsets in development and learning timelines. An organoid that learned a task on day 12 can be compared to one that learned it on day 18. DTW aligns the developmental trajectories, scoring similarity independent of absolute timing. Identify which organoids follow similar developmental paths.
Fleet
Fleet-Scale Organoid Ranking
Rank hundreds of organoids by any behavioral metric: learning onset time, peak synchrony index, response consistency, regime stability, or behavioral complexity (number of distinct regimes). Identify top-performing organoids for downstream experiments, and detect underperformers for protocol adjustment. DBSCAN clustering groups organoids with similar developmental trajectories.

Closed-Loop Control Pipelines

Record, analyze, stimulate, measure. VectorScaleDB serves as the state engine for closed-loop neural control systems.

Pipeline
Record → Analyze → Stimulate → Measure
Each ClosedLoopCycle entity captures one iteration: the recorded neural state, the analysis result (regime classification, anomaly score), the stimulation decision (protocol, parameters), and the measured response. The full cycle history is a temporal vector stream, enabling trajectory analysis on the control loop itself.
Decisions
Regime-Driven Stimulation
Stimulation protocols adapt based on the current neural regime. VectorScaleDB's real-time regime detection classifies the organoid's state (quiescent, bursting, synchronous, learning), and the control system selects the appropriate stimulation protocol. Regime transitions trigger protocol switches within milliseconds.
Gaming
Organoid Gaming State Tracking
For neural gaming paradigms (e.g., organoids playing Pong), GameState entities track the relationship between game state, neural activity, and control output. VectorScaleDB enables queries like "in which game states does the organoid exhibit the highest learning rate?" directly from compressed behavioral data.

Supported Formats & Protocols

Standard
NWB / HDF5
Neurodata Without Borders (NWB 2.x) and raw HDF5 files. Full support for spike trains, LFP, stimulus tables, electrode metadata, and behavioral annotations. NWB's hierarchical structure maps directly to VectorScaleDB's entity model.
Hardware
NEV / NSx
Blackrock Microsystems NEV (neural event) and NSx (continuous) formats. Direct ingestion of multi-electrode recordings from Cerebus and CerePlex systems with automatic channel mapping and threshold detection.
Integration
SpikeInterface / MEA Streams
SpikeInterface-compatible recording objects for integration with Python-based spike sorting pipelines. Real-time MEA stream ingestion for closed-loop applications via WebSocket or gRPC endpoints.
Entity Types
Native Bio-Computing Entity Types
NeuralUnit PopulationState StimulationEvent OrganoidState GameState ClosedLoopCycle
